# AHT10 MakeCode Extension

This extension provides blocks for reading temperature, humidity, dew point and heat index from the AHT10 sensor.

## Blocks

- Temperatur (°C)
- Temperatur (°F)
- Luftfeuchtigkeit (%)
- Taupunkt (°C)
- Heat Index (°C)

## Wiring

AHT10 → micro:bit  
- VCC → 3V  
- GND → GND  
- SDA → P20  
- SCL → P19
